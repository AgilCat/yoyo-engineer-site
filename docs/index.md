## 关于我 {#about}

我是一名全栈 AI 工程师，深耕自然语言处理与计算机视觉领域，擅长从 0 到 1 构建生产级机器学习系统。核心能力覆盖大语言模型微调（SFT/RLHF/DPO）、检索增强生成（RAG）架构设计、多模态模型部署与优化（TensorRT/ONNX/vLLM），以及端到端 MLOps 流水线搭建。

A full-stack AI engineer specializing in NLP and computer vision, with deep expertise in building production-grade ML systems. Core competencies include LLM fine-tuning (SFT/RLHF/DPO), RAG architecture design, multimodal model deployment & optimization (TensorRT/ONNX/vLLM), and end-to-end MLOps pipeline orchestration.

## 项目 {#projects}

### 企业级 RAG 知识引擎

基于向量数据库（Milvus）+ 重排序（Reranker）+ 混合检索架构，构建支持百万级文档的企业知识库问答系统，集成 Agent 工具调用与多跳推理能力。

`RAG` `Milvus` `LangChain` `Agent`

### LLM 推理加速平台

基于 vLLM + TensorRT-LLM 构建高吞吐推理服务，集成 KV-Cache 优化、投机解码（Speculative Decoding）与动态批处理，推理延迟降低 60%。

`vLLM` `TensorRT-LLM` `Speculative Decoding`

### 多模态视觉理解系统

基于 Vision-Language Model（VLM）构建工业质检系统，融合目标检测（YOLOv8）与语义理解，实现缺陷检测 + 自然语言报告生成一体化流水线。

`VLM` `YOLOv8` `多模态`

### 端到端 MLOps 平台

基于 Kubeflow + MLflow 构建全生命周期 ML 平台，覆盖数据版本管理（DVC）、实验追踪、模型注册、A/B 测试与自动化再训练流水线。

`Kubeflow` `MLflow` `DVC` `MLOps`
